    ## WEB-437: Fix default halaman pada DataTable

    <Info>
      **Status:** In Progress\
      **Priority:** No priority\
      **Assignee:** Lalang Trendy Buana\
      **Labels:** Bug, Enhancement\
      **Created:** 2026-01-29\
      **Related:** WEB-436, WEB-415
    </Info>
    ### Context

    #### Problem 1: State Persistence

    Ketika berada dihalaman \> 1 dan klik detail pada record data, kemudian kembali ke tampilan record data sebelumnya otomatis ke halaman 1.

    **Solusi:** Menyimpan halaman terakhir yang dibuka untuk kemudahan user.

    ```javascript
    "stateSave": true
    ```

    #### Problem 2: Filter/Search Reset

    Ketika berada dihalaman \> 1 dan melakukan Filter atau Cari, datatable tidak otomatis reset halaman ke-1 sehingga data tidak ditemukan, padahal hanya berada dihalaman yang salah.

    **Solusi:** Menambahkan kondisi ketika klik tombol Filter atau Cari.

    ```javascript
    $('#list_data').DataTable().state.clear();
    ```
